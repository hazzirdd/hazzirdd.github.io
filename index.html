— My Projects —

GameNight

    Website Demo Github

Utilizing JavaScript, HTML, CSS and PostgresSQL. Hayden Sweet created GameNight. GameNight is a fullstack web app designed to help a party host plan out the boardgames for the group before guests arrive. The host, or guests, can access the host's database of games and select what they want to play! Now the friend with all the games knows what to bring, or the host knows what to pull out for when the party arrives. Games can be added and removed from the virtual table, requested and sorted in a list by multiple categories.


Property Runner

    Demo Github

Inspired by his previous profession, Hayden built out an entire application deticated to the art of property managment. Taking full advantage of the Google Maps API, Python, Flask, SQLAlchemy, Jinja2, JavaScript, HTML and CSS Hayden developed a professional looking and functioning web app to sort, manage and deal with vacancies for any company dealing with properties. Properties can have tasks assigned and marked complete, pictures uploaded and distances marked by an integrated API.
